11
Start tmux ‚Üí launch Claude Code inside it
CRITICAL. Claude Code MUST run inside tmux for split panes.
WSL / BASH ‚Äî RUN IN ORDER
# Step A: Start tmux session

# Step B: Now INSIDE tmux (green bar at bottom)
# Launch Claude Code
claude
‚ö†Ô∏è Order matters! tmux first, THEN claude. Outside tmux = no visible panes.
‚úÖ In tmux? Look for green bar at bottom showing [zonewise].
"sessions should be nested with care"

Already in tmux. Exit first:

BASH
# Detach: Ctrl+B then D
tmux kill-server
tmux new-session -s zonewise
tmux colors broken or garbled
BASH
echo 'set -g default-terminal "screen-256color"' >> ~/.tmux.conf
tmux source-file ~/.tmux.conf
